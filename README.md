# saycheese
Take some screenshots at different screensizes

## Install

    npm install -g https://github.com/orangemug/saycheese.git

## Usage

    $ saycheese --help
    Take some screenshots at different screensizes
    Usage: saycheese

    Examples:
      saycheese -s 400x600 -s 768x1024 http://google.com    


    Options:
      -s, --size    Size of screen, format 768x1024 (multiple allowed)  [required]
      -o, --output  Output to file
